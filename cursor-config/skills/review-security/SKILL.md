# セキュリティレビュー

## 対象

- **git diff** および触れた設定・認証・API。

## 観点

- 認証・認可の抜け（ID直指定、他ユーザーデータ参照）
- 秘密情報のログ・クライアント露出
- XSS/SQLi/コマンド注入の典型パターン
- セッション・CSRF・CORS（該当時）

## 出力

- 重大度付き（Critical / High / Medium / Low）
- 再現または根拠の短文
